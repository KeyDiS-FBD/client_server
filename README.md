# Description

Simple low-level client-server connection. Written in C language

## Compile
Compile source code:
```
make
```
Compile homework code:
```
make home<number>
## Run server
Argument: port and number of clients
```
bin/server 5000 3
```
## Run client
Argument: ip and port.
```
bin/client 127.0.0.1 5000
```
